About Huskify

Huskify is a simple web application for creating a nice looking printable webpage which displays the order of huskies pulling the sledges.

You can quickly add new dogs and sledges, simply by writing the name and pressing enter. After choosing a sledge you can select dogs by clicking, and use drag and drop to organize them in the sledges.

The application is written for Extreme Huskies, using some of the latest web technologies and developent tools: Angular.js, angular-local-storage, Font Awesome, ngDialog, Gulp, PureCSS and etc.

There was two main purposes in bulding this:

  • Learn Angular programming and other new web technologies
  • Ease Extreme Huskies' employers' trouble when organizing sledges
